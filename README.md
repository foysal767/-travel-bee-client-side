Website Name: travelBee

Description About travelBee:

# This is a tourist guide service-related website. Here, you can hire a tourist guide to enjoy the full tour. And give feedback/review about specific services.

# Home page of this website: You can see a navbar, and a hero section, one of the main features of this website is the service section, then another two sections are available here. The footer section is also available on this website. But, in the service section, you can see only 3 services. Bottom of this service section you can see a see all button. If you hit this button you will go service page.

# service page: You can see all services on this page. And the bottom of every service card, see the Details button. If you hit this button it will go a private route. And see all description about service. And another section is the review section on this page. You can see all reviews of this service.

# You can add service and review. And you can delete the review which review was added to you.

# On this website a login page and sign-up page are available. Also, You can continue with google. 

# Uses In this website: React, React-router-dom, firebase, MongoDB, node, express, daisyUi, cors, etc.

# On this website service data and review data get from the MongoDB database.
